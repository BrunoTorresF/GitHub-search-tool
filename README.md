# GitHub Search Tool

Search github for users by location and programming language

## Getting Started

clone the repo

### Prerequisites

webpack ^2.2.1
react ^16.4.1
express^4.16.3
axios ^0.18.0

to install these dependencies we need to run the following command on terminal
```
npm install
```

### Installing

To start the server

```
npm start
```

To start our compiler

```
npm run webpack
```
